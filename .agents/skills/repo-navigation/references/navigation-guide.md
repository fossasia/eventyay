# Navigation Guide

## Fast Path Discovery

- Product code: `app/eventyay/`
- Tests: `app/tests/`
- Deployment: `deployment/`
- Contributor docs: `doc/`, `README.rst`, `CONTRIBUTING.md`

## Common Tasks

- Add backend feature: start in owning domain app under `app/eventyay/`.
- Add API endpoint: use `app/eventyay/api/`.
- Add tests: use `app/tests/`.

## Guardrails

- Treat legacy top-level product folders as reference-only unless explicitly requested.
- Keep new work aligned with current `eventyay.*` module conventions.
