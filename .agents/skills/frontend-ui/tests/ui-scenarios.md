# UI Validation Scenarios

## Scenario 1: Form Submission Error State

- Trigger a validation error.
- Confirm field-level errors are visible and understandable.
- Confirm focus moves to or highlights the problematic field.

## Scenario 2: Mobile Navigation

- Open the page at ~375px width.
- Confirm primary actions remain reachable without horizontal scroll.

## Scenario 3: Keyboard-only Interaction

- Use Tab/Shift+Tab only.
- Confirm all critical controls are reachable and actionable.
