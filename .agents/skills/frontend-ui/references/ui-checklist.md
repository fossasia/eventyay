# Frontend UI Checklist

Use this checklist after making UI changes.

## Behavior

- Interaction works with mouse and keyboard.
- Errors and loading states are visible when relevant.
- No console errors in the browser.

## Accessibility

- Interactive controls have accessible names.
- Form fields are associated with labels.
- Focus order is logical.
- Color contrast is sufficient for key text/actions.

## Responsiveness

- Layout works at common widths (320px, 768px, 1024px, 1440px).
- No clipped controls or overlapping text.
- Tables/lists remain readable on narrow screens.

## Project-specific Conventions

- No inline scripts in templates.
- No jQuery in new code.
- Vue code follows existing Composition API conventions in the target app.
