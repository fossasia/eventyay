# Review Response Scenarios

## Scenario 1: Request for Tests

- Add or adjust tests covering changed behavior.
- Reference exact test names in the review reply.

## Scenario 2: Request for Scope Reduction

- Split unrelated changes into a separate branch/PR.
- Keep this PR focused on one concern.

## Scenario 3: Naming or Style Feedback

- Apply requested naming/style updates.
- Re-run targeted tests and summarize outcomes.
